# Educational Articles

This directory contains educational articles organized by topic.

## Structure

- `eucharist-basics/` - Fundamental teachings about the Eucharist
- `mass-parts/` - Explanations of different parts of the Mass
- `history/` - Historical development of Eucharistic theology
- `living-eucharist/` - Practical guidance for living the Eucharist

## Content Format

All articles must be in Markdown format with theological review metadata:

```markdown
---
title: 'Article Title'
category: 'eucharist-basics'
reviewedBy: 'Reviewer Name'
reviewDate: 'YYYY-MM-DD'
magisteriumRefs: ['CCC 1234-1235']
language: 'en'
---
```

See `docs/content-guidelines/` for complete requirements.
