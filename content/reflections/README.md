# Daily Gospel Reflections

This directory contains daily gospel reflections for the liturgical calendar.

## Format

Each reflection should be in Markdown format with metadata including:

- Gospel reading reference
- Liturgical season
- Theological reviewer
- Reflection date

## Organization

Reflections can be organized by:

- Liturgical year (Year A, B, C)
- Liturgical season (Advent, Christmas, Lent, Easter, Ordinary Time)
- Special feast days
