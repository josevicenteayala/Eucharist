# Media Assets

This directory contains images, videos, and audio files for the platform.

## Organization

```
media/
├── images/          # Static images
├── videos/          # Video content
├── audio/           # Audio files (readings, prayers)
└── icons/           # UI icons and symbols
```

## Guidelines

- Use WebP format for images when possible
- Optimize all media for web delivery
- Include appropriate copyright/attribution
- Maintain accessible alt text for images
- Keep video file sizes reasonable
