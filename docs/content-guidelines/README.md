# Content Guidelines

This directory contains guidelines for creating and reviewing content for the Eucharist Platform.

## Contents

- Content creation standards
- Theological review process
- Style guide
- Magisterium reference requirements
- Multilingual content guidelines

## Key Requirements

All spiritual and educational content must include:
- Theological review metadata
- Magisterium references (e.g., CCC citations)
- Review date and reviewer information
- Language specification

Refer to CONTRIBUTING.md for detailed content requirements.
