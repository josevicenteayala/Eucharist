# Public Assets

This directory contains static files served directly by the web server.

## Contents

- `index.html` - Main HTML template
- `favicon.ico` - Site favicon
- `robots.txt` - Search engine instructions
- `manifest.json` - PWA manifest
- Static images and assets

## Guidelines

- Optimize all assets for web delivery
- Use appropriate image formats (WebP, SVG)
- Keep file sizes minimal
- Include proper meta tags in index.html
