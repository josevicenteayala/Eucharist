# Custom React Hooks

This directory contains custom React hooks for reusable logic.

## Purpose

Extract and share stateful logic across components.

## Examples

- `useAuth.ts` - Authentication state and actions
- `useGospel.ts` - Fetch and manage gospel data
- `useLocalStorage.ts` - Local storage management
- `useMediaQuery.ts` - Responsive design utilities

## Standards

- Prefix all hooks with "use"
- Document hook parameters and return values
- Write unit tests for complex hooks
- Keep hooks focused and composable
