# React Components

This directory contains all React components organized by feature area.

## Organization

```
components/
├── common/          # Shared/reusable components (buttons, forms, layouts)
├── eucharist/       # Eucharist-specific education components
├── gospel/          # Daily gospel reading components
└── community/       # Community feature components
```

## Standards

- Use functional components with hooks
- Implement proper TypeScript typing
- Include accessibility attributes (ARIA)
- Co-locate component tests (*.test.tsx)
- Keep components focused and single-purpose
- Use composition over inheritance
